**Node.js** es un ==entorno de ejecución de código abierto que permite ejecutar código JavaScript fuera de un navegador web, directamente en el servidor o en la computadora del usuario==. Transforma a JavaScript en un lenguaje de propósito general capaz de gestionar bases de datos, lógica de backend y conexiones de red.

## Intalacion
https://nodejs.org/en/download/current

```
# Download and install nvm:
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.5/install.sh | bash

# in lieu of restarting the shell
\. "$HOME/.nvm/nvm.sh"

# Download and install Node.js:
nvm install 26

# Verify the Node.js version:
node -v # Should print "v26.3.0".

# Verify npm version:
npm -v # Should print "11.16.0".
```
