# 📚 Bóveda de Obsidian - Biblioteca de Código

¡Bienvenido a tu biblioteca de conocimiento y notas de desarrollo! Esta carpeta está configurada para funcionar de manera óptima como una bóveda (**Vault**) en **Obsidian**.

---

## 🚀 Cómo Abrir y Usar en Obsidian

Para visualizar y navegar por tus notas con todo el poder de los enlaces internos, sigue estos pasos:

1. **Descarga e Instala Obsidian**: Si aún no lo tienes, descárgalo desde [obsidian.md](https://obsidian.md/).
2. **Abrir Carpeta como Bóveda**:
   - Abre Obsidian.
   - Selecciona la opción **"Open folder as vault"** (Abrir carpeta como bóveda).
   - Busca y selecciona este directorio (`Code`).
3. **Punto de Partida**:
   - Abre la nota **`Welcome.md`** en la raíz. Este es tu **Hub Principal**. Desde aquí podrás navegar con un solo clic a cualquier tema.

---

## 🗺️ Estructura de Navegación

El conocimiento está organizado bajo el concepto de **MOCs (Mapas de Contenido)**:

- **Welcome.md (Hub Principal)**: Enlaza a todos los temas principales.
- **Hubs Temáticos**: Cada carpeta principal (como `Docker`, `Terminal`, `js`, etc.) contiene una nota principal (ej. `JS.md`, `HTML.md`, `Docker.md`) que sirve de índice para todas las notas de esa categoría.
- **Retorno Rápido**: Al final de cada Hub encontrarás un enlace `[[Welcome|⬅️ Volver al Hub Principal]]` para facilitar la navegación fluida sin perderte.

---

## 🛠️ Características Clave en Obsidian

- **Vista de Gráfico (Graph View)**: Presiona `Ctrl + G` (o `Cmd + G` en macOS) para ver una representación visual interactiva de cómo todas tus notas están interconectadas.
- **Enlaces Internos**: Puedes hacer clic en cualquier enlace con formato `[[Ruta/Nota]]` para saltar inmediatamente a ella.
- **Transclusiones**: En carpetas como `Docker` verás bloques del tipo `![[Nota]]`, lo que significa que Obsidian renderizará el contenido de esa nota directamente dentro del documento principal.
